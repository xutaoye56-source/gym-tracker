# Gym Workout Tracker v2 — 健身计划网页

## 目标
基于现有 index.html，增加以下功能。保持单文件 HTML。

## 技术约束（同 v1）
- 单文件 HTML
- 零外部依赖
- localStorage 持久化
- 暗色主题，移动端优先

## 本轮新增功能（按优先级）

### 1. UI 文案优化（零成本）
- 在 subtitle 段落中，localStorage 说明文字后追加：`🔒 你的数据永不离开你的设备`
- 所有「完成打卡」改为「✔ 搞定」
- 所有器械标签用 emoji：器械 → `🔧 器械`，哑铃 → `🏋️ 哑铃`，无需器械 → `🧘 自重`

### 2. 动作卡片视觉层次升级
- 组数×次数用大号加粗显示（如 `3×12` 用 1.2em 粗体，颜色 accent）
- 设备标签用不同底色色块区分（器械蓝色底，哑铃橙色底，自重绿色底）
- keyPoints 和 detail 合并：keyPoints 作为展开前的摘要（显示在卡片上），detail 合并到可展开区域
- 完成的卡片整体降低透明度 + 左边缘绿色竖线

### 3. 自定义重量记录（新增功能）
每个动作卡片上增加一个「重量」输入区：
- 一个小输入框（placeholder: "重量 kg"），数字键盘
- 输入后自动保存到 localStorage，key 格式：`gym-tracker:weight:v1:week:{w}:day:{d}:exercise:{id}`
- 下次打开自动回填上周同动作的重量（如果是第2周）
- 如果上周同动作有记录，在第2周该动作卡片上显示提示：「上周：XX kg」

### 4. 详细动作说明（嵌入完整教学）
每个动作卡片底部增加「📖 怎么做」展开/折叠按钮，展开后显示该动作的完整教学文字。

以下是每个动作的完整教学（按训练分组）：

**训练A动作：**
- lat-pulldown：器械长什么样：一台带高位滑轮的机器，椅子可调高度，上方有长横杆。怎么做：1.大腿垫卡住不漂 2.握距比肩宽一拳，手掌朝前 3.肩膀先下沉，手肘向下向后拉（不是用手拽）4.横杆拉到下巴下方 5.慢放，手臂伸直但肩不耸。最易犯错：身体后仰借力/耸肩/用手臂拽。应感觉到：背阔肌（腋窝下那片）拉扯感，不是手臂酸。
- seated-cable-row：器械长什么样：坐姿划船机，脚踩踏板，面前低位滑轮配V型把手。怎么做：1.脚踩踏板膝微屈 2.先夹紧肩胛骨，手肘贴身体后拉 3.把手拉到腹部，顶峰停1秒 4.慢放。最易犯错：腰前后晃动借力/耸肩/拉太高（到胸口）。应感觉到：背部中间有挤压感。
- chest-press：器械长什么样：靠背椅+两个前推扶手。怎么做：1.座椅调至握把轴心对齐胸中部（最关键）2.后背贴紧，手腕中立 3.肩胛骨收紧贴靠背，前推 4.推到肘微屈不锁死 5.慢收。最易犯错：座椅太高或太低/肩膀前耸/肘关节锁死。应感觉到：胸部撑开和挤压感。
- shoulder-press：怎么做：1.座椅调至握把约在肩高 2.背贴靠背核心收紧 3.向上推至肘微屈 4.下放至下巴高度就停。最易犯错：耸肩/下放太低（低于锁骨）/身体后仰借力。应感觉到：肩膀前侧发热酸胀。若无专门肩推机，用哑铃坐姿推举代替。
- face-pull：器械长什么样：龙门架，滑轮调至面部高度，装绳索把手。怎么做：1.双手各抓绳头，大拇指朝自己 2.手肘向两侧打开与肩同高 3.拉到耳朵两侧 4.慢放。最易犯错：手肘往下拉/太重身体后仰/拉到脖子后。应感觉到：肩后侧和上背灼热感。
- plank：怎么做：1.手肘在肩正下方，前臂贴地 2.脚尖着地，脚跟到后脑勺一条直线 3.腹部收紧（想象有人要打你肚子）4.自然呼吸。最易犯错：塌腰（最危险）/屁股撅太高/憋气。应感觉到：腹部深层收紧，不是下背酸痛。

**训练B动作：**
- leg-press：器械长什么样：大斜板+脚踩板+座椅。怎么做：1.脚距与肩同宽，脚踏板上半部 2.推至腿接近伸直膝不锁 3.下放至膝盖约90度 4.脚后跟发力。最易犯错：膝盖锁死/下放太深屁股离凳/脚太靠下/用手推膝。应感觉到：大腿前侧和臀部发力。
- dumbbell-rdl：怎么做：1.双手各持一轻哑铃（4-6kg起步），站姿脚与髋同宽 2.膝盖微屈，髋部往后推，上身向前倾 3.哑铃贴着腿下滑到小腿中段，背始终挺直 4.臀部发力把髋推回站直。最易犯错：膝盖弯曲太多变深蹲/用腰拉/背弓起。应感觉到：大腿后侧和臀部拉伸感。
- reverse-lunge：怎么做：1.双脚前后站，前脚全掌着地，后脚前掌着地 2.身体垂直下沉，前腿膝盖不超脚尖太多，后膝接近地面不碰 3.前脚后跟发力推回。最易犯错：前膝超脚尖太多/身体前倾/后膝砸地。应感觉到：前腿股四头肌和臀部发力。
- bird-dog：怎么做：1.四足跪姿，手在肩下膝在髋下 2.核心先收紧（最重要），同时伸出右手和左腿 3.伸到身体一条直线，不塌腰不旋转 4.慢放回换边。最易犯错：塌腰旋转/伸太快甩出去/憋气。应感觉到：核心和臀部稳定发力。
- dead-bug：怎么做：1.仰卧，双手伸向天花板，腿抬膝弯90度 2.下背贴紧地面（最重要）3.同时伸出右手和左腿，不碰地 4.回位换边。最易犯错：下背离地/憋气。应感觉到：腹部深层收紧像束腰带。
- ab-crunch：器械长什么样：座椅+胸前垫子/横杆。怎么做：1.脚卡踏板 2.手搭握把不发力 3.纯腹部发力把胸垫往下卷 4.卷到底停1秒慢放。最易犯错：用手臂拉握把/惯性弹/太重。应感觉到：上腹部收紧。

**训练C动作（同训练A/B的复用相同说明，高脚杯深蹲是新增）：**
- goblet-squat：怎么做：1.双手托哑铃一端贴胸口（像高脚杯）2.脚比肩略宽脚尖微朝外 3.先屁股后坐（髋启动）再屈膝 4.蹲到大腿平行 5.脚后跟发力站起。最易犯错：膝盖内扣/先屈膝再屈髋/脚后跟抬起/下背弯曲。应感觉到：大腿前侧和臀部发力，核心全程收紧。

### 5. 数据导出/导入（新增功能）
在页脚区域增加两个按钮：
- 「📥 导出数据」：把所有 `gym-tracker:*` localStorage key 打包成 JSON 下载
- 「📤 导入数据」：读取 JSON 文件覆盖当前 localStorage
